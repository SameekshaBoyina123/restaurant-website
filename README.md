1. Project Overview

The objective of this project is to develop a responsive restaurant website that allows users to view the menu, know about the restaurant, and book a table online. The website is built using HTML, CSS, and JavaScript. 


2. Setup Instructions

Download the project folder
Open index.html in browser
Navigate using menu
Test booking form 

3. Code Structure

index.html – Home page
about.html – About page
services.html – Menu page
contact.html – Booking page
style.css – Design
script.js – Validation 

4. Technical Details

Form validation algorithm:
Read input values
Check empty fields
Validate phone number
Show success message
